# ☕ Meus Projetos e Exercícios de Java

> 📚 Repositório criado com o objetivo de centralizar códigos, exercícios e pequenos projetos desenvolvidos para auxiliar no meu processo de estudo e evolução na linguagem Java.

Este espaço serve como um laboratório prático onde aplico conceitos de Programação Orientada a Objetos (POO), lógica de programação, estruturas de dados e boas práticas de desenvolvimento de software.

---

## 🎯 Objetivo do Repositório

O foco principal deste repositório é didático:
* **Prática Constante:** Fixar a sintaxe e a lógica da linguagem Java através de desafios práticos.
* **Evolução Contínua:** Registrar meu progresso e aprendizado, desde os conceitos mais básicos até estruturas mais avançadas.
* **Documentação:** Servir como um guia de consulta rápida de códigos que eu mesma desenvolvi durante os estudos.

## 🚀 Como Explorar os Códigos

Cada projeto ou exercício está organizado em sua respectiva pasta/pacote. Para testar qualquer um deles:
1. Clone este repositório na sua máquina.
2. Abra a pasta correspondente ao exercício na sua IDE (Eclipse, IntelliJ, VS Code, etc.).
3. Execute a classe que contém o método `main`.
